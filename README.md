# Password Generator (Python + Tkinter)

A simple **Password Generator application** built using **Python** and **Tkinter**.  
This project allows users to generate **strong, secure, and customizable passwords** through a graphical user interface.

---

## Features

- Generate **secure random passwords**
- Select password **length**
- Choose character types:
  - Lowercase letters (a‚Äìz)
  - Uppercase letters (A‚ÄìZ)
  - Digits (0‚Äì9)
  - Symbols (!@#$ etc.)
- Option to **exclude ambiguous characters** (O, 0, l, 1)
- Enforce at least **one character from each selected set**
- Generate **multiple passwords at once**
- Copy selected password or copy all passwords
- Save generated passwords to a **text file**
- Simple and user-friendly GUI
